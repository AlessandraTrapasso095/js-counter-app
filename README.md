# Counter App in JavaScript

Questo progetto è una semplice applicazione web che simula il comportamento di un contatore.
L’utente può aumentare o diminuire il valore del contatore tramite due pulsanti.
L’interfaccia è generata dinamicamente utilizzando JavaScript puro.

## Funzionalità

- Valore iniziale del contatore impostato a 0
- Pulsante "+" per incrementare il valore
- Pulsante "−" per decrementare il valore con limite minimo pari a 0
- Interfaccia creata dinamicamente tramite JavaScript

## Tecnologie utilizzate

- HTML
- CSS
- JavaScript
